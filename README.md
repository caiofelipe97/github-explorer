# github-explorer
Project to explore Github projects and issues

- Functions
  - Search Repository
  - See Repository details and Issues
  - Click into a issue and go to issue page on github
- This project uses github API.
  
# Steps to run:
  - Run "yarn"
  - Run "yarn start"
